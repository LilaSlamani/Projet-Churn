# Projet Data Science : Prédiction de Churn

## Objectif
Prédire les clients susceptibles de résilier leur abonnement ("Churn") afin de mieux cibler les campagnes de fidélisation. 
Ce projet s'inscrit dans une démarche professionnelle de Data Science pour un opérateur télécom.

## Contenu du projet
*   `data/` : Contient le fichier de données `churn_internet.csv`.
*   `scripts/` :
    *   `01_analyse_exploratoire.R` : Nettoyage et statistiques (Gtsummary).
    *   `02_modelisation.R` : Algorithme de prédiction.*
*   `app/` : Code de l'application Web de démonstration.

## Technologies utilisées
*   **Langage :** R
*   **Librairies :**
*   **Modélisation :** Régression Logistique
